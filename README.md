Movie Analytics - это комплексная система реального времени для анализа поведения пользователей стримингового сервиса. Платформа собирает данные о просмотрах фильмов через Kafka, обрабатывает их с помощью Spark Streaming и сохраняет результаты в PostgreSQL.

Система рассчитывает ключевые метрики: активность пользователей, популярность контента, эффективность платформ и устройств. Автоматизированные пайплайны в Airflow обеспечивают оркестрацию процессов от сбора данных до визуализации.

В Superset созданы интерактивные дашборды для мониторинга бизнес-показателей в реальном времени.

Технологический стек: Apache Airflow, Kafka, Spark Streaming, PostgreSQL, Superset, Docker.

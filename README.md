**Movie Analytics Platform** - это высоконагруженная система аналитики в реальном времени для стриминговых видеосервисов. Платформа обрабатывает потоковые данные о просмотрах фильмов и предоставляет детальную аналитику пользовательского поведения.

Архитектура построена на современных Big Data технологиях: Apache Kafka для приема событий просмотров, Spark Streaming для распределенной обработки данных, и PostgreSQL как основное аналитическое хранилище. Система собирает данные с различных клиентских приложений - мобильных, веб и Smart TV.

Оркестрация всего пайплайна данных осуществляется через Apache Airflow, который автоматизирует процессы от проверки здоровья сервисов до генерации финальных отчетов. В реальном времени рассчитываются ключевые бизнес-метрики: DAU/MAU, время просмотра на пользователя, ретеншен и популярность контента.

Для визуализации используется Apache Superset с набором интерактивных дашбордов, отображающих текущую активность пользователей, тренды просмотров и технические показатели системы. Инфраструктура полностью контейнеризована с помощью Docker, что обеспечивает простоту развертывания и масштабирования.

Проект демонстрирует полный цикл работы с потоковыми данными - от приема и валидации событий до сложных агрегаций и бизнес-аналитики. Система способна обрабатывать миллионы событий в час с задержкой менее 30 секунд, предоставляя стриминговому сервису ценнейшие инсайты для оптимизации контентной стратегии и улучшения пользовательского опыта.

**Ключевые технологии:** Apache Airflow, Kafka, Spark Streaming, PostgreSQL, Superset, Docker, Python, Celery.

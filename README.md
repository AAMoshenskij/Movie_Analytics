Movie Analytics - это высоконагруженная система аналитики в реальном времени для стриминговых видеосервисов. Платформа обрабатывает потоковые данные о просмотрах фильмов и предоставляет детальную аналитику пользовательского поведения.

Архитектура построена на современных Big Data технологиях: Apache Kafka для приема событий просмотров, Spark Streaming для распределенной обработки данных, и PostgreSQL как основное аналитическое хранилище.

Оркестрация всего пайплайна данных осуществляется через Apache Airflow, который автоматизирует процессы от проверки здоровья сервисов до генерации финальных отчетов. В реальном времени рассчитываются ключевые бизнес-метрики: время просмотра на пользователя и популярность контента.

Для визуализации используется Apache Superset с набором интерактивных дашбордов, отображающих текущую активность пользователей, тренды просмотров и технические показатели системы. Вот примеры графиков и таблиц. Инфраструктура полностью контейнеризована с помощью Docker, что обеспечивает простоту развертывания и масштабирования.

Проект демонстрирует полный цикл работы с потоковыми данными - от приема и валидации событий до сложных агрегаций и бизнес-аналитики. 

**Ключевые технологии:** Apache Airflow, Kafka, Spark Streaming, PostgreSQL, Superset, Docker, Python, Celery.
